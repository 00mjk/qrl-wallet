Build Instructions
==================

#### Requirements

- Nodejs 8.x / npm
- Meteor
- Git


## Windows

`npm install -g qrl-electrify`

`cd C:\path\to\qrl-wallet`

`cd .electrify`

`npm install qrl-electrify`

`cd ..`

`electrify package -o ./.electrify/.dist`



## OSX / Ubuntu

`npm install -g qrl-electrify`

`cd /path/to/qrl-wallet`

`cd .electrify`

`npm install qrl-electrify`

`cd ..`

`electrify package -o ./.electrify/.dist`
